# SentimentAnalyzer

This is a simple sentiment analysis web application.

## Installation

**Frontend:**

No specific installation is required for the frontend. Simply open `index.html` in your web browser.

**Backend:**

1. Make sure you have Node.js and npm installed.
2. Navigate to the `backend` directory.
3. Run `npm install express cors @huggingface/inference` to install the necessary dependencies.
